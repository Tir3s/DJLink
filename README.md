#+ DJLink

Crowd-interactive DJ request platform built with Flutter. Audiences can send tipped song requests and shoutouts in real time while DJs manage a prioritised queue, events, and moderation tools.

DJ Link is a Flutter-based mobile app that connects DJs, event organisers, and audiences at live events. Audience members can submit song requests with optional tips and shoutouts, see the live status of their requests, and follow their favourite DJs, while DJs get a prioritised request queue, simple event management, basic analytics, and moderation tools - all designed for dark nightclub environments and real-time interaction.
